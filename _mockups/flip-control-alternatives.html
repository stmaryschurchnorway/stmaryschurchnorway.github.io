6. Spacer: "Front | Back"

👔
Front | Back
Front View
✓ Shows both options at once
✓ Compact
✓ Like tab switches
✗ Doesn't show arrow/action
✗ Less clear it's clickable
✗ Looks more like a label

Recommendation

Option 3 (Context-Sensitive Labels): "← Back · Front"
This shows the user where they are (Front) and where they can go (← Back). It's compact, intuitive, and updates dynamically. When flipped, it becomes "Back · Front →" — always showing current state + direction.
Second choice: Option 2 (Arrow Only)
If you want the absolute cleanest design and trust users will understand the flip icon (or add a tooltip on hover). Works great on mobile where space is limited.
