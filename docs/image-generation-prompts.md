# Image Generation Prompts for Liturgy Cards

## Reference Style
All prompts should match the aesthetic of the existing **Holy Qurbana** card:
- Warm, golden-lit Orthodox church interior
- Multiple robed clergy in cream/white and ornate vestments
- Liturgical elements: altar, candles, incense, religious iconography
- Formal, reverent, sacred atmosphere
- Professional photography/illustration quality
- Malankara Orthodox Syrian Church tradition
- Dimensions: ~400×300px for card display

## Notes for Image Generators

- **Style Consistency:** All images should match the warm, reverent aesthetic of the Qurbana card
- **Liturgical Accuracy:** Preserve authentic Malankara Orthodox Syrian Church traditions, vestments, and ceremonial elements
- **Color Palette:** Primary colors: cream, gold (#d4a017), maroon (#8B1E2D), white, warm yellows
- **Lighting:** Warm, golden church lighting with candles and architectural illumination
- **Composition:** Professional religious photography with depth, clear focal points, and reverential framing
- **Detail Level:** Rich detail on vestments, liturgical objects, and church architecture
- **Dimensions:** Optimize for 400×300px card displays (may be cropped slightly)
- **Quality:** Professional, high-fidelity images suitable for a respect-centered religious website
